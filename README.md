# SQL Sales Data Analysis

## Project Overview
This project analyzes sales data using SQL to answer business questions and demonstrate SQL skills.

## Tools Used
- PostgreSQL
- pgAdmin 4
- Visual Studio Code
- Git
- GitHub

## SQL Concepts Covered
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- INNER JOIN
- LEFT JOIN
- CTE (Common Table Expressions)
- Window Functions
- Aggregate Functions

## Project Structure

```
SQL_Sales_Data_Analysis
│
├── dataset
├── sql_queries
├── screenshots
└── README.md
```

## Business Questions Solved
- Total sales by customer
- Top-selling products
- Category-wise sales
- Highest spending customers
- Running total of sales
- Customer ranking using Window Functions
